# Browser Extension Icons

The extension requires icons in the following sizes:
- 16x16 pixels (icon16.png) - Browser toolbar
- 48x48 pixels (icon48.png) - Extension management page  
- 128x128 pixels (icon128.png) - Chrome Web Store

Create simple icons with a clipboard or resume theme. You can use any image editor or online icon generator.

For now, you can create simple placeholder icons or use existing icons from your system.

Example: Create a simple clipboard icon with "RT" text for Resume Tracker.

You can temporarily use any 16x16, 48x48, and 128x128 PNG files and name them accordingly.
